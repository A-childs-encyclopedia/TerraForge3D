# TerraGen3D
3D Procedural Terrain Generation Tool in OpenGL/C++

# What can this do?

* Generte 3D Terrain Procedrally using Perlin Noise
* Export Terrain mesh as OBJ
* You can write and test your own shaders!
* Test under different lighting!
* A basic 3D viewer!
